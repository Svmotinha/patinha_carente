#Patinha Carente

O **Patinha Carente** é uma plataforma digital desenvolvida para conectar animais resgatados a lares amorosos. Criado para apoiar a ONG Patinha Carente, 
o sistema moderniza e simplifica o processo de adoção, dando maior visibilidade aos cães e gatos que aguardam por uma nova família. 

Através desta aplicação, os usuários podem visualizar os pets disponíveis, conhecer suas histórias, perfis e necessidades, facilitando o encontro entre o adotante ideal e o animalzinho.

## 🚀 Tecnologias Utilizadas
* **Backend:** Java, Spring Boot, Maven
* **Frontend:** TypeScript, Angular
* **Banco de Dados:** MySQL
